# Arjun Bajpai — Portfolio
This is the frontend code. Backend is available <a href='https://github.com/Arjunslab/portfolio-backend'>here</a>

Personal portfolio website built with React + Tailwind CSS featuring NOVA, an AI assistant that answers questions about me.

## Tech Stack
- React + Vite
- Tailwind CSS
- Axios

## Features
- Glassmorphism / futuristic UI
- Skills & Projects showcase
- NOVA AI chatbot

## Run Locally
```bash
cd portfolio-frontend
npm install
npm run dev
```
